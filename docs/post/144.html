5. copyparty v1.19.7

Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

Copyparty 是一个文件共享工具，提供多种下载选项和功能更新。最新版本 v1.18.9 修复了 CVE-2025-54796 漏洞，v1.15.0 默认禁用上传去重功能，v1.14.3 修复了可能导致数据丢失的 bug。

新功能包括 chdir 选项用于更改工作目录。bug 修复包括改进空文件夹作为状态文件存储的使用，以及优化快速浏览文件夹的体验。其他变更包括 Docker 镜像的元数据修复和状态文件路径调整。

推荐下载 copyparty-sfx.py，仅需 Python 即可运行。其他选项包括英文版、Docker 镜像、Windows 可执行文件等，部分版本存在安全隐患。Python 包可在 PyPI 获取。项目还提供 Discord 社区和演示服务器。

6. fd v10.3.0

A simple, fast and user-friendly alternative to 'find'

新增功能：添加隐藏选项--mindepth作为--min-depth的别名。

日期处理变更：替换humantime和chrono为jiff库，主要影响日期相关选项：
- 月份单位从"M"改为"mo/mos/month/months"
- 月/年时间计算改为基于实际日历天数

- 新增aarch64 Windows的CI和发布支持

其他：更新多个依赖项。
