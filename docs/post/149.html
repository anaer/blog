2. dify 1.9.0

Production-ready platform for agentic workflow development.

Dify 1.9.0 引入两大核心功能：知识管道和基于队列的图引擎。

知识管道采用模块化架构，支持可视化节点编排，提供文档处理的完整工作流。主要特性包括：
1. 支持多样化数据源（本地文件/在线文档/云盘/爬虫）和工具插件
2. 新增问答处理器和图像提取功能
3. 提供测试运行和调试支持
4. 支持DSL导入导出模板
5. 一键迁移旧版知识库

基于队列的图引擎改进工作流执行：
1. 采用统一队列调度模型
2. 支持任意节点启停的灵活执行
3. 新增流处理组件ResponseCoordinator
4. 提供外部控制命令机制
5. 可扩展的GraphEngineLayer插件层

升级注意事项：
1. 必须执行数据源凭证迁移命令
2. 提供Docker和源码两种升级方式
3. Beta版用户需注意数据清除风险

未来计划将增强调试工具、智能调度、多模态支持等功能。该版本包含200+项代码改进，重点提升稳定性和可扩展性。

Microsoft PowerToys is a collection of utilities that help you customize Windows and streamline everyday tasks

提供了四种安装包及其SHA256校验值：
- 用户级x64安装包：PowerToysUserSetup-0.94.2-x64.exe (CE665...B003)
- 用户级ARM64安装包：PowerToysUserSetup-0.94.2-arm64.exe (1AD77...B55)
- 系统级x64安装包：PowerToysSetup-0.94.2-x64.exe (A0C2A...BB8B)
- 系统级ARM64安装包：PowerToysSetup-0.94.2-arm64.exe (2E59B...AF20)

4. mihomo v1.19.14

A simple Python Pydantic model for Honkai: Star Rail parsed data from the Mihomo API.

本次更新主要包含以下内容：

功能新增：
1. Tun模式支持直接发送ping请求
2. 为客户端和服务端添加mTLS支持
3. 支持mieru 0-RTT握手协议
4. 移除了hy1/hy2/tuic出站的ca和ca-str选项，改用fingerprint
5. 支持AmneziaWG v2.0
6. 为Tun添加disable-icmp-forwarding选项
7. 为SS客户端/服务器支持kcptun插件

系统兼容性说明：
- macOS用户需注意：Go 1.25不再支持macOS 11，建议根据系统版本选择对应标签的二进制文件
- amd64平台已弃用-amd64和-amd64-compatible版本，改用-amd64-v1/v2/v3标记CPU级别

BUG修复：
1. 修复了服务器多路复用连接未关闭的问题
2. 解决了多个NTP时间同步相关的问题
3. 修复了vision读取过滤不完整的问题
4. 修复了tuic服务器goroutine泄漏问题
5. 解决了xudp服务器源地址丢失问题
6. 修复了多个数据竞争和内存泄漏问题
7. 解决了多个协议实现中的细节问题

维护更新：
1. 清理和优化了多个模块的代码
2. 更新了依赖库
3. 改进了内存使用效率
4. 优化了多个协议的实现细节
5. 增强了代码的防御性编程
6. 简化了多个功能模块的实现

完整变更日志可查看项目GitHub页面。
