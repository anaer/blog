9. CloudflareSpeedTest v2.3.2

🌩「自选优选 IP」测试 Cloudflare CDN 延迟和速度，获取最快 IP ！当然也支持其他 CDN / 多个解析 IP 的网站 ~

  • Windows（x86、x86_64、ARM v8）

特别提醒：

  1. Windows_arm64版本仅适用于ARM架构设备

  3. Android手机多为ARM架构，需自行测试32/64位
