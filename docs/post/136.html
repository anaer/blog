5. CloudflareSpeedTest v2.3.4

🌩「自选优选 IP」测试 Cloudflare CDN 延迟和速度，获取最快 IP ！当然也支持其他 CDN / 多个解析 IP 的网站 ~

本次更新内容主要包括：

  1. 更新依赖版本，最低编译版本改为Go 1.18
  2. 修复Windows系统下彩色文字显示异常问题
  3. 调整主界面蓝色文字为青色，提升可读性

已知问题：

  • Windows系统通过Git Bash运行时，测速进度条数字颜色异常（因第三方库pb与color的兼容性问题）
  • Windows系统特定环境下进度条刷新异常（终端缓冲导致换行显示）

建议Windows用户使用Windows Terminal等现代终端工具避免兼容性问题。

下载地址：

  • 国内推荐蓝奏云：https://xiu.lanzoub.com/b0742hkxe
  • 提供多平台版本：包括macOS（分新旧系统版本）、Linux（支持多种架构）、Windows（分新旧系统版本）
  • 特殊说明：
    • 旧系统版本使用Go v1.20编译，可能存在问题
    • 路由器用户需根据架构选择对应版本
    • Android/iOS手机需确认架构版本

使用提示：

  • Linux解压命令：tar -zxf cfst_linux_XXX.tar.gz
  • macOS使用方法与Linux相同
