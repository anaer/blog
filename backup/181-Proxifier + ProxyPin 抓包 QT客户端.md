目标: 抓包HTTPS接口, 又不影响SOCKET接口

## Proxifer

配置文件(P) -> 代理服务器(P)...
地址: 127.0.0.1
端口: 9099
协议: HTTPS

配置文件(P) -> 代理规则(R)...
新增代理规则, 目标尽量缩小范围, 因为有些三方服务不允许代理, 可能导致各种问题
名称: New
应用程序: test.exe; test1.exe
目标主机: *.test.com; *.example.com
目标端口: 80;443
动作: Proxy HTTPS 127.0.0.1

其他未说明的直接默认配置

## ProxyPin
HTTPS代理 -> 安装证书到本系统, 选择"受信任的根证书颁发机构"
设置 -> 代理 取消"设置为系统代理" 取消"SOCKS5"
设置 -> 域名代理列表 -> 代理域名黑名单
