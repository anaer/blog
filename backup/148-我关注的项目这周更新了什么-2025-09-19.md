
# 1. [datafaker 2.5.0](https://github.com/datafaker-net/datafaker/releases/tag/2.5.0)  
Generating fake data for the JVM Java, Kotlin, Groovy has never been easier!

- 更新发布文档并设置自动发布功能

- 创建新的Credentials凭证提供程序(原Internet方法已弃用)

- 金融提供程序更新：信用卡格式、Swift IBAN标准(v87→v99)
- 星际迷航提供程序新增星际飞船内容

3. 问题修复：
- 修复CSV生成限制问题
- 修复韩国kpop.boy_bands键名拼写错误
- 修复不稳定的WeightedRandomSelector测试
- 修复巴西本地部分问题及测试格式
- 避免日志中出现无用的NoSuchMethodRuntimeException

4. 代码优化：
- 重构BaseFakerTest及其扩展测试
- 移除YAML文件中的重复项
- 减少测试日志输出
- 消除递归提供程序调用
- 移除无用的System.out.println

5. 数据改进：
- 优化韩国姓名数据
- 修正韩语文件中的拼写错误

6. 新贡献者：
- 欢迎5位新贡献者加入项目

版本从2.4.4更新至2.5.0。

# 2. [copyparty v1.19.9](https://github.com/9001/copyparty/releases/tag/v1.19.9)  
Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

Copyparty是一个文件共享工具，最新版本v1.19.8修复了CVE-2025-58753漏洞（单文件分享权限检查问题）。重要更新包括：
- v1.15.0默认禁用上传去重功能
- v1.14.3修复了可能导致数据丢失的bug

新特性：
- Windows/MacOS/FAT32/NTFS支持模拟大小写敏感行为（可通过casechk:n禁用）
- 支持Unicode文件名/路径的模糊搜索（需手动启用srch-icase）
- 新增--qr-stdout和--qr-stderr参数

修复的bug：
- 基础上传器不接受空文件
- Windows去重功能对远古时间戳文件失效
- WebDAV对未映射根目录的错误响应
- Markdown编辑器多余HTTP请求问题

其他改进：
- 修复图片查看器快捷键错误
- 优化波兰语翻译
- 改进systemd脚本
- 下载选中文件时生成更合理的zip文件名

下载推荐：
- 最佳选择：copyparty-sfx.py（跨平台Python脚本）
- 备选：copyparty-en.py（仅英语版）、Docker镜像
- Windows专用：copyparty.exe（Win8+）、u2c.exe（CLI上传工具）
- 不推荐：copyparty32.exe（仅限Win7本地使用）

该项目提供Discord社区用于重要更新通知，并设有只读演示服务器。
