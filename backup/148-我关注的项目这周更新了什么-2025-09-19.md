# 5. [copyparty v1.19.11](https://github.com/9001/copyparty/releases/tag/v1.19.11)  
Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

Copyparty 是一个文件共享工具，提供多种下载选项。最新版本 v1.19.8 修复了 CVE-2025-58753 漏洞（单文件共享权限检查问题）。此外，v1.15.0 默认禁用上传去重功能，v1.14.3 修复了可能导致数据丢失的 bug。

推荐下载选项：
- **copyparty-sfx.py**：最佳选择，仅需 Python 即可运行
- copyparty-en.py：仅英文版，无多语言支持
- Docker 镜像：适合偏好 Docker 的用户
- copyparty.exe：适用于 Win8 及以上系统，内置缩略图生成
- u2c.exe：命令行上传工具，支持 Win7+

不推荐选项：
- copyparty32.exe：仅适用于 Win7，存在安全隐患
- cpp-winpe64.exe：仅适用于 64 位 WinPE 系统

其他信息：
- 提供只读演示服务器：https://a.ocv.me/pub/demo/
- 设有 Discord 服务器用于重要更新通知
- Python 包可在 PyPI 获取
- 最新修复了 FTP 服务器根目录未映射的 bug (#827)
