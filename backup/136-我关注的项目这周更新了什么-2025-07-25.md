
# 1. [chsrc v0.2.2](https://github.com/RubyMetric/chsrc/releases/tag/v0.2.2)  
chsrc 全平台通用换源工具与框架. Change Source everywhere for every software

chsrc v0.2.2版本更新摘要：

这是第三个重大版本更新，主要改进包括：支持精准测速、提供deb包和安卓预编译文件、项目升级至C11标准、引入chsrc-bootstrap支持更多平台、优化开发工具链和文档。

换源功能改进：使用rawstr4c简化多个recipe，修复Flutter/Arch Linux CN/uv等换源问题，新增Windows下rustup换源支持，所有target支持ls命令。

源配置调整：移除部分网易和搜狐镜像源，更新Cargo源，移除Go的华为源，新增多个上游默认源和测速链接。

新增5个国内镜像站：西安交通大学、哈尔滨工业大学、齐鲁工业大学、大连东软信息学院开源镜像站，并重新启用重庆大学镜像站。

新增4位首次贡献者：@xrgzs、@sanchuanhehe、@Mikachu2333和@techoc。
