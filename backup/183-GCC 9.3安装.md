## 安装依赖包

```sh
yum install wget make gmp-devel mpfr-devel libmpc-devel bzip2 zlib-devel
```

## 下载源码包

```sh
wget https://ftp.gnu.org/gnu/gcc/gcc-9.3.0/gcc-9.3.0.tar.gz
tar xf gcc-9.3.0.tar.gz
cd gcc-9.3.0
```

## 下载依赖项并编译

```sh
./contrib/download_prerequisites
mkdir build
cd build
../configure --enable-bootstrap --enable-shared --enable-threads=posix --enable-checking=release --with-system-zlib --enable-languages=c,c++
make -j 8
make install
```

## 验证版本

```sh
gcc -v
```
