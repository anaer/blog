Linux 中安装字体

查看系统中的字体

fc-list

查看系统中的中文字体

fc-list :lang=zh

将字体文件拷贝到/usr/share/fonts/中 

cp simsun.ttc /usr/share/fonts  
cd /usr/share/fonts  
mkfontscale  
mkfontdir

检验：

fc-list :lang=zh

就可以看到刚安装的字体了

Ubuntu

apt-get install fontconfig  
apt-get install xfonts-utils

CentOS

yum -y install fontconfig #安装字体库  
yum -y install ttmkfdir mkfontscale #安装字体索引信息
