索引管理

新增索引

可以在新增索引时, 自定义分词器custom_pattern 

PUT {{myes}}/my_index  
{  
  "settings": {  
    "number_of_shards": 1,  
    "number_of_replicas": 1,  
    "analysis": {  
      "analyzer": {  
        "custom_pattern": {  
          "type": "pattern",  
          "pattern": "\\|"  
        }  
      }  
    }  
  },  
  "mappings": {  
    "properties": {  
      "title": {  
        "type": "text",  
        "analyzer": "standard"  
      },  
      "content": {  
        "type": "text",  
        "analyzer": "standard"  
      },  
      "timestamp": {  
        "type": "date"  
      }  
    }  
  }  
}

  1. Simple Tokenizer
  2. 描述：简单分词器，根据非字母数字字符进行分词，适用于多语言文本。主要特点是根据短横线、中横线、下划线、逗号、冒号、分号等常见符号进行分词。它不进行任何大小写转换或其他复杂处理。
  3. 示例
    {  
  "analyzer": "simple",  
  "text": "中文|汉字|测试"  
}  

{  
  "tokens": [  
    {  
      "token": "中文",  
      "start_offset": 0,  
      "end_offset": 2,  
      "type": "word",  
      "position": 0  
    },  
    {  
      "token": "汉字",  
      "start_offset": 3,  
      "end_offset": 5,  
      "type": "word",  
      "position": 1  
    },  
    {  
      "token": "测试",  
      "start_offset": 6,  
      "end_offset": 8,  
      "type": "word",  
      "position": 2  
    }  
  ]  
}
  1. Whitespace Tokenizer
  2. 描述：根据空格进行分词，忽略标点符号。

  1. Keyword Tokenizer
  2. 描述：保持输入的文本作为单个分词。
  3. 示例：
    {  
  "analyzer": "keyword",  
  "text": "Hello, this is a document to be analyzed."  
}  

{  
  "tokens": [  
    {  
      "token": "Hello, this is a document to be analyzed.",  
      "start_offset": 0,  
      "end_offset": 41,  
      "type": "word",  
      "position": 0  
    }  
  ]  
}
  1. Pattern Tokenizer
  2. 描述: 正则分词器, 创建索引时指定规则
  3. 示例
