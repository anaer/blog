### 🚀 功能特性

#
### ✨ 升级优化

* 修改 ohscript 生成随机 uuid 方式提升性能  
#
## Elsa 图形编辑平台
#
### 🚀 功能特性
#
#### 核心框架 (elsa)

* 图形具备统一的抽象数据结构。
   * Graph、Page、Shape 结构通用于任意业务场景。
   * 全链路 JSON 序列化能力，兼容任何持久化存储方案。

* 支持拖拽式流程编排。
   * 支持图形拖拽和画布拖拽，并提供节点整理和一键显示所有节点的能力。
#
#### 前端应用 (elsa-react)

* 集成 React 的能力。
   * 基于 Context 的上下文传递机制。
   * 节点渲染缓存：React.memo + 自定义 shouldComponentUpdate。

* 集成 Ant Design 的能力。
   * 基于 Form 组件的实时校验提示系统。
   * 基于 Tree 组件封装的节点上下文观察者机制。
