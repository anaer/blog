
### 安装zsh

```sh
# Ubuntu/Debian
sudo apt install zsh -y
# macOS（默认已安装）
zsh --version
```

#### 设置为默认shell

```sh
chsh -s $(which zsh)
```

### 安装[Oh My Zsh](https://github.com/ohmyzsh/ohmyzsh)

```sh
sh -c "$(curl -fsSL https://install.ohmyz.sh)"
```
