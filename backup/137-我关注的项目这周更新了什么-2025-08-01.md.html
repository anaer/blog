1. hurl 7.0.0

Hurl, run and test HTTP requests with plain text.

Hurl 7.0.0 版本于2025年7月28日发布，感谢多位贡献者的参与。该版本包含多项重要更新：

主要变更：
- 新增 replaceRegex 过滤器并修复 replace 过滤器不再支持正则表达式

功能增强：
- 新增 HTTP 重定向查询功能
- 新增 urlQueryParam 过滤器
- 在长错误格式中显示 curl 命令
- 支持配置每个请求的最大超时时间
- 新增日期比较谓词
- 添加 pinnedpubkey 命令行选项
- 新增 base64 URL 安全编解码过滤器
- 支持解析 curl 的 --cookie 标志
- 为帮助信息添加彩色显示
- 新增 toHex 过滤器
- 添加 first 和 last 过滤器
- 移除 hex crate 依赖
- 改进测试进度输出显示
- 支持 nth 过滤器的负值参数
- 新增 UUID 验证谓词
- 改进捕获错误的显示信息
- 新增 NTLM 和协商认证选项

问题修复：
- 修复 POST 重定向时的 curl 命令错误
- 修复 curl 命令中无效头参数的问题
- 修复秘密变量覆盖问题
- 修复 zsh 自动补全功能
- 修复 HTML 报告中的标签问题
- 修复重定向时的请求体问题
- 支持 JSONPath 中的负索引

安全修复：
- 修复 HTML 报告中通过正则表达式实现的 JavaScript 注入漏洞
