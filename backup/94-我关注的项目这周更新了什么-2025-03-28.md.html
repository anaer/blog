5. docs v3.0.0

A collaborative note taking, wiki and documentation platform that scales. Built with Django and React. Opensource alternative to Notion or Outline.

变更说明

⚠️ Breaking Changes

We are not using the nginx auth request anymore to access the collaboration server (yProvider). The authentication is now managed directly from the yProvider server. You must remove the annotation nginx.ingress.kubernetes.io/auth-url from the ingressCollaborationWS in your helm files.

This means as well that the yProvider server must be able to access the Django server. To do so, you must set the COLLABORATION_BACKEND_BASE_URL environment variable to the yProvider service. Example here: https://github.com/suitenumerique/docs/commit/544dd00c16313197b22debb29ff5cdbd99452de1
