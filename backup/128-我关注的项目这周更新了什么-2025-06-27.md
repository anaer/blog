
# 1. [cherry-studio v1.4.6](https://github.com/CherryHQ/cherry-studio/releases/tag/v1.4.6)  
🍒 Cherry Studio is a desktop client that supports for multiple LLM providers.
本次更新主要包含功能优化、问题修复和代码重构三个方面：

功能优化：
1. 新增对.doc文件的处理支持
2. 允许删除非活跃主题
3. 新增MCP服务器的思考字段
4. 上下文菜单增加拼写检查和字典建议
5. 知识库显示更新时间戳
6. 更新API密钥管理界面
7. 实现Python MCP服务器
8. 支持npm配置自定义注册地址
9. 添加早期访问功能开关

问题修复：
1. 修复API检查参数错误
2. 禁用主题重命名时的思考状态
3. 修复Markdown中LaTeX括号处理
4. 改进数据配置
5. 修复非流式输出问题
6. 清除搜索缓存
7. 修复VLLM错误请求
8. 修复代码编辑器保存问题

代码重构：
1. 文件操作重构为FileAction服务
2. 代码预览自动调整边栏
3. 重命名isWindows为isWin保持一致性
4. 重构数据迁移模态逻辑
5. 优化消息滚动条样式
