# 5. [mihomo v1.19.11](https://github.com/MetaCubeX/mihomo/releases/tag/v1.19.11)  
A simple Python Pydantic model for Honkai: Star Rail parsed data from the Mihomo API.

本次更新主要包含以下内容：

功能新增：
1. DNS客户端新增支持`disable-ipv4`、`disable-ipv6`、`skip-cert-verify`、`ecs`和`ecs-override`参数
2. 转换器新增支持anytls/socks/http协议
3. 为tun设备新增`loopback-address`支持

安全更新：
1. 配置文件中的所有路径将被限制在workdir目录下，可通过设置`SAFE_PATHS`环境变量添加额外安全路径
2. RESTful API中`/configs`的"path"参数受到限制
3. 移除了`proxy-groups`中对`routing-mark`和`interface-name`的支持，需在`proxies`中直接指定

DNS解析调整：
1. UDP部分的DNS解析延迟到连接发起阶段
2. 不同出站类型采用不同的解析逻辑

问题修复：
1. 修复了chacha20计数器溢出、HTTPS转换、SSR URI解码等多项问题
2. 修复了字符串分割返回值检查、远程目标解析等多个错误

维护优化：
1. 代码清理和重构
2. 新增速率限制支持
3. 优化DNS回复写入方式
4. 允许通过环境变量`SKIP_SYSTEM_IPV6_CHECK`跳过系统IPv6检查

注：workdir默认位置为Unix系统的`$HOME/.config/mihomo`或Windows的`%USERPROFILE%/.config/mihomo`
