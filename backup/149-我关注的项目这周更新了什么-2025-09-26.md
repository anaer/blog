# 2. [dify 1.9.0](https://github.com/langgenius/dify/releases/tag/1.9.0)  
Production-ready platform for agentic workflow development.

Dify 1.9.0 引入了两大核心功能：知识流水线和基于队列的图引擎。

知识流水线采用模块化设计，提供可视化节点编排界面，支持：
1. 灵活组合文档处理流程，支持快速启用官方模板和自定义DSL导入导出
2. 通过插件市场集成多样化数据源（本地文件/在线文档/云盘/爬虫）和处理工具
3. 新增Q&A分块策略和图片提取功能，支持混合文本图像输出
4. 提供单步调试能力，可查看中间变量和Markdown预览
5. 支持一键迁移旧版知识库

基于队列的图引擎改进包括：
1. 统一队列调度模型，优化并行任务管理
2. 支持任意节点启动执行和流式响应协调
3. 新增命令机制实现外部控制（暂停/终止）
4. 通过GraphEngineLayer插件层扩展监控能力

升级注意事项：
- 必须执行数据源凭证迁移命令
- 容器部署需备份数据后升级
- 源码部署需更新依赖并运行迁移脚本

未来计划将增强调试工具、智能调度、多模态支持等功能。该版本主要聚焦架构改进，包含200+项代码修复和优化，涉及安全增强、性能调优和UI改进等方面。
