# 3. [hurl 7.0.0](https://github.com/Orange-OpenSource/hurl/releases/tag/7.0.0)  
Hurl, run and test HTTP requests with plain text.

Hurl 7.0.0 版本于2025年7月28日发布，主要更新内容如下：

感谢所有贡献者，包括ecolinet、theofger等18位开发者。

重大变更：
- 新增replaceRegex过滤器，并修复replace过滤器不再支持正则表达式的问题

功能增强：
- 新增HTTP重定向查询功能
- 新增urlQueryParam过滤器
- 长格式错误输出时显示curl命令
- 支持配置每个请求的最大超时时间
- 新增日期比较谓词
- 新增pinnedpubkey命令行选项
- 新增Base64 URL安全编解码过滤器
- 解析curl的--cookie标志
- 为hurl --help添加彩色显示
- 新增toHex、first/last等过滤器
- 移除hex依赖项
- 改进测试进度输出
- 支持nth过滤器的负值和模板参数
- 新增isUuid谓词
- 改进捕获错误信息
- 新增NTLM和协商认证选项

错误修复：
- 修复POST重定向的curl命令错误
- 修复hurlfmt对无效curl头参数的校验
- 修复curl命令的verbose标志解析
- 修复秘密变量覆盖问题
- 修复zsh补全功能
- 修复cookie过期日期解析
- 替换已弃用的libxml2函数
- 修复HTML报告中的注释行显示
- 修复重定向时的请求体问题
- 修复GraphQL HTML导出中的变量标记
- 支持jsonpath负索引

安全修复：
- 修复HTML报告中通过正则表达式字面量导致的JavaScript注入漏洞

用户可通过官方博客了解新版本详情。

# 5. [higress v2.1.6](https://github.com/alibaba/higress/releases/tag/v2.1.6)  
🤖 AI Gateway AI Native API Gateway

Higress 2.1.5版本发布摘要

核心更新：
1. 新增31项更新，包含13项新功能、5项缺陷修复、7项重构优化、5项文档更新和1项测试改进
2. 两项重点功能：
   - Higress API MCP服务器：支持通过MCP协议管理路由、服务源和插件，提升AI Agent对Higress资源的管理能力
   - WASM Go插件迁移：改用原生Go 1.24编译，替代TinyGo，提升插件性能和兼容性

关键特性：
1. Higress API MCP服务器：
   - 通过Higress控制台API实现配置管理
   - 支持路由/服务源/插件的增删改查操作
   - 需在ConfigMap中配置控制台URL及认证信息

2. WASM插件升级：
   - 升级Go版本至1.24.4
   - 移除TinyGo依赖，简化构建流程
   - 提升运行时效率和安全性

其他重要更新：
1. 新增AI代理插件功能：
   - 支持Azure OpenAI三种URL格式
   - 增加Anthropic/Gemini API支持
   - 实现Qwen模型文本重排序功能

2. 系统优化：
   - 修复Envoy内存泄漏问题
   - 优化路由重定向逻辑
   - 放宽DNS域名验证规则

Higress Console更新：
1. 12项更新（6新功能/5修复/1重构）
2. 核心改进：
   - 支持单路由配置多个子路由
   - 新增插件JSON Schema定义
   - 实现系统配置撤销/重做功能
   - 修复LLM提供商管理UI问题

版本统计：
- Higress：31项变更（2项重大更新）
- Console：12项变更

致谢所有贡献者的工作！
