
# 1. [winmerge v2.16.50.2](https://github.com/WinMerge/winmerge/releases/tag/v2.16.50.2)  
WinMerge is an Open Source differencing and merging tool for Windows. WinMerge can compare both folders and files, presenting differences in a visual text format that is easy to understand and handle.

WinMerge 2.16.50.2 是一个稳定版本，推荐用户升级。主要更新内容包括：

文件比较方面修复了显示大二进制文件比较结果时的崩溃问题；文件夹比较修复了过滤器仅应用于第一个路径的bug；过滤器对话框修复了安装或创建新过滤器后的崩溃问题；更新了巴西葡萄牙语、简体中文、法语、意大利语等8种语言翻译。

2.16.50版本新增功能包括：最大化窗口时点击标题栏顶部边缘将视为点击标签按钮；修复了未翻译字符串问题；更新了巴西葡萄牙语、简体中文、匈牙利语等7种语言翻译。

2.16.49 Beta版本主要改进包括：支持鼠标滚轮切换标签页；文件比较修复复制多个差异块时的崩溃问题；文件夹比较改进过滤系统支持表达式；更新7-Zip到25.00版本；安装程序优化；更新了巴西葡萄牙语、简体中文、法语等14种语言翻译。

已知问题包括：选项窗口可能导致INI文件损坏、大文件比较时可能崩溃、目录差异大时比较速度慢等。

# 6. [fd v10.3.0](https://github.com/sharkdp/fd/releases/tag/v10.3.0)  
A simple, fast and user-friendly alternative to 'find'

本次更新主要包含以下内容：

1. 新增功能：
- 为`--min-depth`添加了隐藏别名`--mindepth`

2. 重要变更：
- 用`jiff`crate替换了`humantime`和`chrono`，影响日期相关选项如`--changed-within`和`--changed-before`：
  - 'M'不再表示"月"(month)，需改用"mo"/"mos"/"month"/"months"
  - 月份和年份计算现在考虑日历差异而非固定秒数

3. 其他改进：
- 新增aarch64 Windows的CI和发布工件
- 更新多个依赖项
- 改进对Illumos系统的构建支持

4. 兼容性说明：
- 这将是最后一个支持x86_64 Mac OS的测试版本
- 可能是最后一个支持x86_64 Mac和Windows7的Rust版本
