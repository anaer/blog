# 3. [winmerge v2.16.50.2](https://github.com/WinMerge/winmerge/releases/tag/v2.16.50.2)  
WinMerge is an Open Source differencing and merging tool for Windows. WinMerge can compare both folders and files, presenting differences in a visual text format that is easy to understand and handle.

WinMerge 2.16.50.2 稳定版发布，主要更新内容如下：

1. 修复了文件比较中"仅显示比较结果"功能在处理大型二进制文件时崩溃的问题。

2. 修复了文件夹比较中筛选器只应用于第一个文件夹路径的问题。

3. 修复了筛选器对话框在安装或创建新筛选器后崩溃的问题。

4. 包含多个语言翻译更新：巴西葡萄牙语、简体中文、法语、意大利语、韩语、立陶宛语、葡萄牙语和俄语。

其他改进包括：
- 2.16.50版本新增了最大化窗口时点击标题栏顶部的处理优化
- 2.16.49 Beta版本新增了鼠标滚轮切换标签页功能
- 改进了文件夹比较的筛选系统
- 更新7-Zip至25.00版本
- 修复了多个已知问题

已知问题包括：
- 比较大型文件时可能崩溃
- 比较差异较大的目录时速度较慢
- 粘贴文本后垂直滚动条可能出现问题

此版本推荐所有用户升级，以取代之前的稳定版本。

# 5. [copyparty v1.19.7](https://github.com/9001/copyparty/releases/tag/v1.19.7)  
Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

Copyparty是一个自托管文件共享工具，提供多种下载选项。最新版本v1.18.9修复了CVE-2025-54796拒绝服务漏洞。主要特性包括新增`chdir`选项用于更改工作目录，修复了空文件夹作为状态文件存储的问题，优化了文件夹快速滚动功能。

推荐下载`copyparty-sfx.py`，仅需Python即可运行；也提供Docker镜像和Windows可执行文件（注意32位版本仅限Win7且不安全）。其他选项包括英文专用版`copyparty-en.py`、CLI上传工具`u2c.exe`等。

项目维护活跃，设有Discord服务器通报重要更新。最新版本在荷兰RevSpace社区发布。历史版本中，v1.15.0默认禁用上传去重功能，v1.14.3修复了可能导致数据丢失的严重bug。

Docker相关改进包括修复镜像仓库元数据、将`$HOME`改为`/state`以避免配置文件错误。用户可通过演示服务器https://a.ocv.me/pub/demo/体验功能。

# 6. [fd v10.3.0](https://github.com/sharkdp/fd/releases/tag/v10.3.0)  
A simple, fast and user-friendly alternative to 'find'

本次更新主要包含以下内容：

功能改进：
- 新增隐藏参数`--mindepth`作为`--min-depth`的别名

核心变更：
- 用`jiff`库替换了`humantime`和`chrono`库，影响日期相关参数如`--changed-within`和`--changed-before`的格式：
  - 月份单位不再使用"M"（避免与分钟混淆），改用"mo/mos/month/months"
  - 月份和年份的计算现在考虑实际日历天数而非固定秒数
- 新增对aarch64 Windows平台的CI测试和发布支持
- 更新了大量依赖库
- 改进了对Illumos系统的构建支持

其他说明：
- 这将是最后一个支持x86_64架构Mac OS的测试版本（因GitHub将停止该硬件支持）
- 可能是最后一个支持x86_64 Mac和Windows 7的Rust版本（对应Rust将降级支持）
