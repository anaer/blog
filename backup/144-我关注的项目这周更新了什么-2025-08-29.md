
# 1. [winmerge v2.16.50.2](https://github.com/WinMerge/winmerge/releases/tag/v2.16.50.2)  
WinMerge is an Open Source differencing and merging tool for Windows. WinMerge can compare both folders and files, presenting differences in a visual text format that is easy to understand and handle.

WinMerge 2.16.50.2稳定版发布，作为推荐版本替代早期版本。主要更新包括：

1. 2.16.50.2修复内容：
- 文件比较：修复大二进制文件"仅显示比较结果"时的崩溃问题
- 文件夹比较：修复过滤器仅应用于第一个文件夹路径的问题
- 过滤器对话框：修复安装或创建新过滤器后的崩溃问题
- 更新多国语言翻译（巴西、简体中文、法语等9种语言）

2. 2.16.50主要改进：
- 通用：优化最大化窗口时标题栏边缘点击处理
- 修复未翻译字符串问题
- 更新巴西、简体中文等7种语言翻译

3. 2.16.49 Beta版重要变更：
- 新增鼠标滚轮切换标签页功能
- 文件比较：修复复制多个差异块时崩溃问题
- 文件夹比较：改进过滤系统，支持表达式和UI增强
- 更新7-Zip至25.00版本
- 更新15种语言翻译

已知问题：
- 选项窗口操作可能导致INI文件损坏
- 大文件比较时可能崩溃
- 差异较大目录比较速度慢
- 垂直滚动条在粘贴文本后可能异常

该版本包含多项功能改进和错误修复，建议用户升级。

3. **平台变更**：
   - 停止提供amd64和amd64-compatible版本，改为按CPU级别（v1/v2/v3）区分amd64版本

4. **问题修复**：
   - 修复了VLESS加密、Vision协议、自动重定向等方面的问题
   - 解决了文件更新覆盖、macOS签名、数据丢失等多个稳定性问题
   - 修复了ALPN禁用、WireGuard连接等网络相关功能

5. **维护更新**：
   - 优化自动更新接口，支持通道和强制更新参数
   - 升级Go语言版本至1.25及相关依赖库
   - 重构加密代码，改进类型安全性和性能
   - 完善测试用例，清理冗余代码

完整变更日志可查看GitHub版本对比。用户需注意及时更新自动升级脚本以适配新版本。

# 5. [copyparty v1.19.7](https://github.com/9001/copyparty/releases/tag/v1.19.7)  
Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

Copyparty 是一个文件共享工具，提供多种下载选项和功能更新。最新版本 v1.18.9 修复了 CVE-2025-54796 漏洞，v1.15.0 默认禁用上传去重功能，v1.14.3 修复了可能导致数据丢失的 bug。

新功能包括 `chdir` 选项用于更改工作目录。bug 修复包括改进空文件夹作为状态文件存储的使用，以及优化快速浏览文件夹的体验。其他变更包括 Docker 镜像的元数据修复和状态文件路径调整。

推荐下载 `copyparty-sfx.py`，仅需 Python 即可运行。其他选项包括英文版、Docker 镜像、Windows 可执行文件等，部分版本存在安全隐患。Python 包可在 PyPI 获取。项目还提供 Discord 社区和演示服务器。

# 6. [fd v10.3.0](https://github.com/sharkdp/fd/releases/tag/v10.3.0)  
A simple, fast and user-friendly alternative to 'find'

新增功能：添加隐藏选项`--mindepth`作为`--min-depth`的别名。

日期处理变更：替换`humantime`和`chrono`为`jiff`库，主要影响日期相关选项：
- 月份单位从"M"改为"mo/mos/month/months"
- 月/年时间计算改为基于实际日历天数

平台支持：
- 新增aarch64 Windows的CI和发布支持
- 改进Illumos系统构建支持
- 最后一次支持x86_64 Mac OS的版本（因GitHub停止该架构支持）
- 可能是最后一个支持x86_64 Mac和Windows 7的Rust版本

其他：更新多个依赖项。
