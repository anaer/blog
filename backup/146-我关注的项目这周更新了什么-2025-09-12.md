# 3. [copyparty v1.19.8](https://github.com/9001/copyparty/releases/tag/v1.19.8)  
Portable file server with accelerated resumable uploads, dedup, WebDAV, FTP, TFTP, zeroconf, media indexer, thumbnails++ all in one file, no deps

copyparty 是一个自托管的文件共享工具，最新版本修复了 CVE-2025-58753 漏洞，该漏洞可能导致共享文件夹中的其他文件被访问。主要更新内容包括：

1. 安全修复：
   - 修复了单文件共享权限检查缺失的问题
   - 加强了密码提示的安全性

2. 新功能：
   - 新增 IdP 登录/登出重定向选项
   - 可根据用户权限隐藏磁盘使用情况和服务器版本信息
   - 添加共享创建权限控制选项
   - 文件夹删除文件后实时更新总大小

3. 改进与优化：
   - 支持非 QWERTY 键盘布局
   - FTP 服务器兼容性改进
   - 多项性能优化
   - Unix 域套接字权限修复

4. 下载选项：
   - 推荐 copyparty-sfx.py（跨平台 Python 版本）
   - 提供 Docker 镜像、Windows 可执行文件等
   - Win7 专用版本存在安全风险

5. 其他：
   - 提供 Discord 社区
   - 包含演示服务器链接
   - 详细版本历史记录

该项目持续更新，建议用户升级到最新版本以确保安全。
