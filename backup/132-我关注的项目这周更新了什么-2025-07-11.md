# 10. [CloudflareSpeedTest v2.3.2](https://github.com/XIU2/CloudflareSpeedTest/releases/tag/v2.3.2)  
🌩「自选优选 IP」测试 Cloudflare CDN 延迟和速度，获取最快 IP ！当然也支持其他 CDN / 多个解析 IP 的网站 ~

本次更新修复了调试模式下因重定向地址出错导致下载测速失败时显示地址不准确的问题，现在能正确区分是原始地址还是重定向地址出错。

提供了多平台多架构的下载链接，包括：
- MacOS（x86_64/ARM v8）
- Linux（x86/x86_64/ARM系列/Mips系列）
- Windows（x86/x86_64/ARM v8）

特别提醒：
1. 国内用户建议从蓝奏云下载
2. Windows ARM64版本仅适用于ARM架构设备
3. 路由器用户可通过命令判断架构，推荐尝试Mips/Mipsle/ARM版本
4. Android手机多为ARM架构，需自行测试32/64位版本
5. Linux系统解压命令：tar -zxf
6. MacOS系统需要通过命令行运行

附：
- OpenWrt固件用户可尝试原生编译版本
- IOS用户需根据虚拟机架构选择对应版本
